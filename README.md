# Firebase Contact App

## Overview

The Firebase Contact App is a simple web application built with React.js and Firebase Firestore. It allows users to manage their contacts by adding, updating, and deleting them.

## Setup

1. Clone the repository:-
   ```bash
   git clone <repository_url>
   cd firebase-contact-app

   
Install dependencies:-
Copy code
npm install


Configure Firebase:-
Create a Firebase project and enable Firestore.
Replace the placeholder Firebase configuration values in config/firebase.js with your actual configuration.


Start the development server:-
npm start


Functionality:-
View a list of contacts
Search for contacts by name
Add new contacts
Update existing contacts
Delete contacts


Components:-
Navbar: Displays the application's navigation bar.
ContactCard: Represents a contact card with options to edit or delete the contact.
AddAndUpdateContact: Modal component for adding or updating contacts.
NotFoundContact: Component displayed when no contacts are found.
